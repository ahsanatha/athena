How this system works?

Untuk versi Alpha saat ini kami menggunakan rules-based system untuk menentukan prioritas setiap track tugas akhir. Untuk setiap track telah di definisikan beberapa mata kuliah yang mendukung untuk track tersebut.


Rules ini di didefinisikan dengan mempertimbangkan diagram kurikulum 2020.

What Next?

Dengan bertambahnya data di database kami, kami dapat membuat model yang menggeneralisir pertimbangan prioritas dengan menggunakan pendekatan machine learning.
